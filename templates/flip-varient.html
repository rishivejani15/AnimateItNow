Flip Card Effects



CODE

Horizontal Flip

FRONT
BACK
<div class="flip-card" onclick="this.classList.toggle('flipped')">
  <div class="card-front">FRONT</div>
  <div class="card-back rotateY">BACK</div>
</div>

.flip-card.flipped {
  transform: rotateY(180deg);
}
.card-back.rotateY {
  transform: rotateY(180deg);
}
